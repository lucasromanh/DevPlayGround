## ğŸ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS)
- [NPM](https://www.npmjs.com/)

### Installation

1. **Clone and Navigate**
   ```bash
   git clone https://github.com/lucasromanh/DevPlayGround.git
   cd DevPlayGround
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Launch the Engine**
   ```bash
   npm run dev
   ```

Explore the app at `http://localhost:3000`.
